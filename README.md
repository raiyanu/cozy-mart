CozyMart is an e-commerce website built using React and Framer Motion. It utilizes CSS modules for styling.

## Installation

To get started with CozyMart, follow these steps:

1. Clone the repository:

   ```bash
   git clone https://github.com/raiyanu/cozy-mart.git
   ```

2. Navigate to the project directory:

   ```bash
   cd cozy-mart
   ```

3. Install the dependencies using Yarn:

   ```bash
   yarn
   ```

   or using npm:

   ```bash
   npm install
   ```

### Usage

To run CozyMart locally, use the following commands:

- Using Yarn:

  ```bash
  yarn dev
  ```

- Using npm:

  ```bash
  npm dev
  ```

This will start the development server and you can access the website at `http://localhost:6123`.

### Deployment

To deploy CozyMart to Netlify, follow these steps:

1. Create an account on [Netlify](https://www.netlify.com/) if you haven't already.

2. Connect your GitHub repository to Netlify.

3. Be careful while authorizing GitHub repos to Netlify.

4. Configure the build settings:

   - Build command: `yarn build` or `npm run build`
   - Publish directory: `build`

5. Click on "Deploy site" and Netlify will build and deploy your CozyMart website.
